调用API格式
https://<Cisco_IPICS_Server>/ipics_server/services/NorthboundService/<function>
[?<parameter>=<value>][&<function>[?<parameter>=<value>]...]

对应的参数含义：
 <Cisco_IPICS_Server> IP地址或服务器名
 <function> 想要调用的API函数
 <parameter> 函数参数
 <value> 函数参数的值
